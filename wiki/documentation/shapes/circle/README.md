# Two.Circle

### Constructor


| Argument | Description |
| ---- | ----------- |
| `x` | The x position of the circle. |
| `y` | The y position of the circle. |
| `radius` | The radius value of the circle. |
| `resolution` | The number of vertices used to construct the circle. |
